## ⚙️ **Como Executar**  

### **Pré-requisitos**  
- Java JDK 17+  
- Maven (opcional)  

### **Passo a Passo**  

1. **Clone o repositório**  
   ```bash
   git clone https://github.com/luyz-gusta/MVC-Java
   cd MVC-Java
   ```

2. **Compile e execute**  
   ```bash
   javac src/main/java/Main.java
   java -cp bin Main
   ```
